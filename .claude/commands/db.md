# Database

Run Prisma commands from the Back-End directory.

## Migrate
```
cd Codebase/Back-End && npx prisma migrate dev
```

## Generate client
```
cd Codebase/Back-End && npx prisma generate
```

## Seed
```
cd Codebase/Back-End && npx prisma db seed
```

## Studio (visual DB browser)
```
cd Codebase/Back-End && npx prisma studio
```
